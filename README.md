# 🎬 Netflix Analytics Dashboard

An interactive data analytics dashboard built to explore Netflix's content library through powerful visualizations and insights. This project helps analyze content distribution, genre popularity, ratings, release trends, and country-wise content availability.

## 📊 Features

* Content Distribution Analysis
* Genre-wise Content Insights
* Movie vs TV Show Comparison
* Country-wise Content Availability
* Release Year Trend Analysis
* Rating Distribution Analysis
* Top Contributing Countries
* Interactive Filtering and Slicing
* Data-Driven Visualizations
* Business Intelligence Reporting

## 📈 Key Insights

* Identified the most popular genres on Netflix
* Analyzed content growth over the years
* Compared Movies and TV Shows across multiple dimensions
* Explored country-wise content contributions
* Examined audience rating distributions
* Uncovered trends in content production and release patterns

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* SQL
* Power BI
* Data Visualization

## 📂 Dataset

The dataset contains information about Netflix titles, including:

* Title Name
* Content Type
* Genre
* Country
* Release Year
* Rating
* Duration
* Date Added
